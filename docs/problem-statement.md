# FAIR LENS Problem Statement

Hiring systems can encode bias through historical data, proxy variables, and uneven model outcomes across demographic groups.

FAIR LENS addresses that gap by providing:

- dataset ingestion for CSV and JSON files
- automatic sensitive attribute detection
- a fairness audit contract for dashboarding and reporting
- a backend structure that separates routes, services, models, and configuration

The goal is to make bias detection and fairness reporting visible early in the workflow so teams can review datasets before deployment.
